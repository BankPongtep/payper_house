# Payper House

A Leasing and Hire Purchase Management System built with Laravel 11, React 18, and Docker.

## Features

- **Authentication**: Role-based login (Owner/Customer).
- **Owner Dashboard**: Manage assets, customers, and contracts.
- **Customer Dashboard**: View contracts and payment status.
- **Contract Management**: Calculate installments, interest, and generate schedules.
- **Modern UI**: Built with React, Tailwind CSS, and Lucide Icons.

## Tech Stack

- **Backend**: Laravel 11 (PHP 8.2)
- **Frontend**: React 18 (Vite)
- **Database**: MySQL 8.0
- **Containerization**: Docker & Docker Compose

## Getting Started

1. **Clone the repository**
2. **Start Docker**:
   ```bash
   docker-compose up -d --build
   ```
3. **Access the Application**: `http://localhost:8000`
